# Test Publish Formula

This repository is used to test the process of publishing a formula

## Intallation

```bash
brew install ourworldcenter/repo/test-hello
```

## Usage

Print out "Hello World!" or "Hello _Name_" if a name is provided.

```bash
test-hello
// Hello World!

test-hello "John"
// Hello John!
```
